palette-shader

GitHub

A WebGL shader (Three.js) that visualizes a color palette by mapping a color space onto a plane or polar wheel, then snapping each pixel to the nearest palette color via a configurable distance metric. The six views above show the same palette across X, Y and Z slices — first in polar (cylinder), then in linear (cube) mode.

Install

npm install palette-shader

Usage

Notes

Requires Three.js as a peer dependency. Color models and distance metrics are orthogonal — the model controls how the color space is rendered, the metric controls which palette color each pixel snaps to.

The typeface is the open-source Iosevka. Created by David Aerne.
